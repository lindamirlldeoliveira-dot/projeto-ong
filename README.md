# Projeto ONG Esperança Viva – Site Institucional

Este repositório contém um site institucional estático desenvolvido para uma ONG fictícia, como parte de um trabalho da faculdade. O site apresenta informações institucionais, missão, projetos e formas de contato.

---

## Visão Geral

- **Projeto**: Site institucional de ONG  
- **Objetivo**: Criar uma presença online simples para uma ONG fictícia, com páginas para apresentar a organização, seus serviços e valores  
- **Deploy**: Hospedado no GitHub Pages  
- **URL do site**: [Projeto ONG](https://lindamirlldeoliveira-dot.github.io/projeto-ong/)

## Controle de Versão (Git / GitHub)

- Foi utilizada a **estratégia GitFlow**, com branches organizadas para garantir um desenvolvimento estruturado:  
  - `main`: versão estável e pronta para deploy  
  - `develop`: branch principal de desenvolvimento  
  - `feature/…`: branches para novas funcionalidades  
  - `release/…`: preparação de versões para entrega  
  - `hotfix/…`: correções urgentes  

- As mensagens de commit seguem um **padrão semântico** para manter o histórico claro e compreensível:  
  - `feat:` para novas funcionalidades  
  - `fix:` para correções de bugs  
  - `style:` para ajustes visuais  
  - `docs:` para mudanças na documentação  
  - `refactor:` para melhoria do código sem alterar a funcionalidade  
  - `chore:` para tarefas de manutenção  

## Acessibilidade (WCAG 2.1 – Nível AA)

Para garantir que o site seja acessível, foram seguidas algumas diretrizes da **WCAG 2.1 – nível AA**:
 
- HTML semântico: uso de `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>` para estruturar o conteúdo  
- Contraste de cores: cores de texto e fundo com contraste adequado para legibilidade  
- Leitores de tela: uso de `alt` em imagens e `aria-label` quando necessário para descrever elementos interativos  
- Tema alternativo: pensando em modos de maior contraste / escuro para usuários com necessidades visuais variadas
